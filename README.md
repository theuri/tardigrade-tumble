# 🐻 Tardigrade Tumble

A tiny browser game: guide a **tardigrade** (water bear) hopping over holes in the moss across **20 increasingly hard levels**.

▶️ **Play:** https://theuri.github.io/tardigrade-tumble/

No install — runs in any modern browser, on phone or computer.

## How to play
- **Jump:** `Space` / `↑` / `W` / tap
- **Pause:** `Esc` or `P` · **Restart:** `R`
- **Mute:** `M` · **Music:** `N`

## Features
- 20 procedurally-built levels that get faster, with wider and tighter gaps
- Collect glowing spores for a **1–3 star** rating per level
- Unlock **colors** and **hats** as you progress
- **Realistic** (microscope water-bear) graphics by default; unlock a **Cartoon** toggle after clearing 10 levels (`G`)
- Unlock a **Translucent** skin after clearing all 20 (`T`)
- Upbeat chiptune music + sound effects

Single self-contained `index.html` — no dependencies, no build step.
